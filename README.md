# Glass Login Form

A modern and responsive login page built using pure HTML and CSS.

## Features

- Modern Glassmorphism Design
- Fully Responsive Layout
- Clean and Professional UI
- Gradient Background
- Smooth Hover Effects
- Beginner Friendly Code
- No JavaScript Required

## Technologies Used

- HTML5
- CSS3

## Preview

This project demonstrates how to create a stylish login form using only HTML and CSS while maintaining a clean and responsive user experience.

## Project Structure

```text
glass-login-form/
├── index.html
├── style.css
└── README.md
```

## Getting Started

1. Clone the repository.
2. Open `index.html` in your browser.
3. Customize the design as needed.
